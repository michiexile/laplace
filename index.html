Graph Laplacian webtoy

Uses d3.js and Numeric Javascript. Largely based on modifications of rkirsling's example code.

Click in the open space to add a node, drag from one node to another to add an edge.
Ctrl-drag a node to move the graph layout. Click a node or an edge to select it. When it is selected, Delete removes it.
